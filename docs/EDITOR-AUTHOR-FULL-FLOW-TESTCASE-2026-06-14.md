## 二、测试目标

验证编辑和作者围绕同一个选题完成从 SI 策划、预发、转项目、阶段稿件提交与审核、正文审稿修订、全文质检、项目完成到终稿导出的完整业务闭环。

本用例重点验证：

- 编辑端可创建 SI 并预发给绑定作者。
- 作者端只能查看被预发的 SI，不能编辑、收回或转项目。
- 编辑可基于 SI 预发记录确认转项目，系统自动创建项目并进入梗概阶段。
- 作者按项目阶段提交梗概、细纲、正文、全文质检。
- 编辑按 Doc 状态审核通过或退回，退回时必须填写说明。
- 编辑审稿第一章时可产生批注、编辑建议、删除修订和新增修订。
- 作者可查看编辑审稿结果，修改后重新提交。
- 全部章节通过后，编辑可解锁全文质检。
- 全文质检通过后，编辑可标记项目完成并导出终稿。

## 六、关键状态校验

| 阶段 | 校验对象 | 预期状态 |
| --- | --- | --- |
| SI 创建后 | SI | `draft` 或保存并预发后为 `prereleased` |
| SI 预发后 | SI 预发记录 | `active`；作者端“我的 SI”可见 |
| 转项目后 | SI | `converted`；不可重复预发或重复转项目 |
| 转项目后 | 项目 | 生命周期 `active`；当前阶段 `synopsis` |
| 梗概通过后 | 项目 | 当前阶段推进到 `outline` |
| 细纲通过后 | 项目 | 当前阶段推进到 `chapter` |
| 第一章退回后 | 第一章 Doc | `returned`；持有人 `author`；作者待办/通知产生 |
| 第一章重新提交后 | 第一章 Doc | `submitted`；持有人 `editor` |
| 第一章通过后 | 第一章 Doc | `approved` |
| 第二章通过后 | 第二章 Doc | `approved`；全部章节通过 |
| 质检解锁后 | 全文质检 Doc | 从 `locked` 变为可进入的草稿态 |
| 全文质检提交后 | 全文质检 Doc | `submitted`；持有人 `editor` |
| 全文质检通过后 | 全文质检 Doc | `approved` |
| 项目完成后 | 项目 | 生命周期 `completed`；项目只读 |
| 终稿导出后 | 下载文件 | 产生 DOCX 文件，内容来自全文质检终稿 |

## 七、权限与页面入口校验

| 校验点 | 编辑预期 | 作者预期 |
| --- | --- | --- |
| 导航权限 | 可见“SI 选题策划库”“SI 预发记录”“我的项目”“AI 助手”等 | 可见“我的 SI”“我的项目”；不可见 SI 管理和预发记录 |
| SI 操作 | 可新建、编辑、预发、转项目 | 只能查看被预发 SI |
| Doc 提交 | 编辑持稿时主要执行退回/通过 | 作者持稿时可编辑、保存、提交 |
| 审稿修订 | 编辑持有 Doc 时可批注、编辑建议、标记删除、直接新增修订 | 作者查看退回稿时可看到编辑审稿结果并修改正文 |
| 质检解锁 | 章节全部通过后可解锁 | 不可手动解锁 |
| 项目完成 | 全文质检通过后可标记完成 | 不可标记完成 |
| 终稿导出 | completed 项目仍可导出终稿 | 作者端不应出现编辑/管理员导出项目终稿权限 |

## 八、通过标准

本用例满足以下条件时判定通过：

- 42 个步骤均能按顺序完成，无权限错乱、按钮缺失、状态异常或页面不可恢复错误。
- 每次提交、退回、通过后，Doc 状态、持有人、待办和通知符合预期。
- 第一章退回前，编辑审稿至少包含 1 条批注、1 条编辑建议、1 处删除修订、1 处新增修订。
- 作者重新打开第一章退回稿时，可以看到退回说明和所有编辑审稿标记。
- 两章正文均审核通过后，质检解锁按钮由禁用变为可用。
- 全文质检通过后，编辑可以完成项目。
- completed 项目为只读状态，但“导出项目终稿”仍可用。
- 导出的 DOCX 可以成功下载，且终稿内容优先来自全文质检 Doc。

## 九、失败与缺陷记录建议

| 失败场景 | 记录重点 |
| --- | --- |
| 找不到作者测试账号 | 记录绑定关系、预发弹窗作者列表、接口错误信息 |
| 保存并预发失败 | 记录必填字段、主类型、适配作者、错误提示 |
| 转项目失败 | 记录预发记录状态是否为 `active`、是否已转换、错误提示 |

### 第一章正文初稿

夜雨落在龙城南门的石阶上，巡夜人发现了一封被血浸透的密信。信封没有署名，只压着半枚盐运司旧印，像是有人故意把线索留在最显眼的地方。

沈知微赶到时，尸体已经被衙役抬走，只剩墙角一串断开的脚印。她认出那种鞋底纹路出自城西马帮，而马帮三日前刚替盐运司运过一批账册。

她没有立刻声张，只把密信收入袖中。信纸背面透出一行极淡的字：账在明月楼，人在旧案里。

### 第二章正文初稿

明月楼白日闭门，入夜才点灯。沈知微借送药之名进入后厨，在柴房暗格里找到一本缺了封皮的暗账。

暗账第一页只写了三个名字，其中一个正是三年前旧案中被处斩的证人。她意识到，龙城命案不是新的祸端，而是有人在清理旧案留下的人证。

楼外忽然响起马蹄声，谢云峥推门而入，低声提醒她：“盐运司的人到了，你只有半炷香。”
